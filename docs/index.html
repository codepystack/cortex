Cortex

An open AI platform with a blazing-fast Rust backend and a modern SvelteKit interface.

⭐ View on GitHub 🤝 Contribute

Features

🦀

Rust / Axum Backend

High-performance, memory-safe REST API with JWT authentication and streaming support.

⚡

SvelteKit Frontend

Reactive UI built with SvelteKit and Tailwind CSS — fast by default.

🔌

OpenAI-Compatible API

Drop-in replacement endpoint for chat completions with streaming (SSE).

🤖

Multi-Provider Models

Works with OpenAI, Anthropic, and a built-in cortex-echo demo model.

🔑

Auth & JWT

Register/login flow with secure JWT-based session management.

🛠️

Agents & Tools

Extensible registry for models, agents, tools, workflows, and MCP servers.

Quick Start

# 1. Clone the repo
git clone https://github.com/codepystack/cortex.git
cd cortex

# 2. Start the backend
cd backend
cp .env.example .env   # add your API keys (optional)
cargo run              # → http://localhost:8080

# 3. Start the frontend (in a new terminal)
cd frontend
npm install
npm run dev            # → http://localhost:3000

Tech Stack

Rust 1.78+ Axum 0.8 Tokio SvelteKit 2 Svelte 5 Tailwind CSS 4 TypeScript JWT SSE Streaming
